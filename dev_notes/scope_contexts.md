## out of scope

These contexts are outside the scope of this project.

1. Cop: Replaced by Cop2.
2. Shop: Replaced by Mat (Vop based).
3. Data: Abstract nodes used to represent data structures within data flows.
4. Manager, Director: Special nodes (def, dir, root, etc.) that manage the root-level environment (such as `/`, `/obj`).
5. VopNet, CopNet, ChopNet, TopNet: Containers storing nodes in a specific context (VOP, COP, CHOP, TOP).
